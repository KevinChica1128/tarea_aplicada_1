\section{Introducción}
Se construyó el tipo de documento \textit{revcoles} para edición de artículos para la Revista Colombina de Estadística (www.estadistica.unal.edu.co/revista), posteriormente se amplió para los documentos a ser sometidos al Simposio de Estadística, incluyendo la opción \textit{symposium}. Luego se adicionó la opción \textit{project} para los trabajos de grado de la Carrera de Estadística y finalmente la opción \textit{report} para los trabajos e informes de cursos.

Este documento tiene comentadas algunas partes con el entorno \textit{comment}, si se desean usar basta borrar o comentar \verb"begin{comment}" y \verb"end{comment}" del comienzo y final de la parte respectiva. Algunas son: título en inglés, resumen, resumen en inglés.

El formato está diseñado para imprimir los informes a dos caras en
hojas tamaño carta.

\section{Formato del texto}
Para utilizar esta plantilla se graba con un nombre nemotécnico que el o los estudiantes deseen y en la carpeta determinada para contener los archivos asociados para el trabajo. Sobre ese archivo se reemplaza este contenido por el del trabajo o informe.

En el directorio de trabajo, además de la plantilla, deben estar los archivos: \emph{revcoles.cls}, el cual no se debe modificar; \emph{references.bib}, en el cual el autor debe introducir los datos de las referencias en formato \BibTeX\ y los archivos de las gráficas, por ejemplo \emph{graph\_example.eps} para la gráfica insertada en este documento.

El archivo \emph{revcoles.cls} provee el formato completo para los
artículos a ser publicados en la Revista Colombiana de Estadística%
\footnote{http://www.ciencias.unal.edu.co/publicaciones/estadistica/rce/index.html}.
En el formato se incluye la opción \emph{report}:

\verb"\documentclass[report]{revcoles}"

para editar los trabajos e informes de cursos.

\subsection{Autores}

\section{Citas bibliográficas} Para las \textbf{Referencias} se
utiliza el paquete
Harvard\footnote{http://tug.ctan.org/tex-archive/macros/latex/contrib/harvard/harvard.ps},
formato autor-año. Estas deben realizarse con el programa BibTeX
el cual requiere que las referencias sean grabadas en un archivo
de extensión \emph{.bib} \cite[pp. 205--215]{Castro-03}.

En la sección referencias solamente deben aparecer las citadas en el
texto. Las citas dentro del texto se introducen mediante el comando:
\verb"\cite{clave}"

Por ejemplo: \cite{Caballero-86}, \cite{Dodge-85},
\cite{Conover-81}, \cite{Searle-92}.

Si el nombre del autor forma parte del texto, el comando utilizado es: \verb"\citeasnoun{clave}". Por ejemplo:

\begin{itemize}

\item Según el trabajo de \citeasnoun{Borges-05}\dots

\item Los modelos propuestos por \citeasnoun{Dodge-85}\dots

\item Tal como lo indica \citeasnoun{Conover-81} en su trabajo de \dots

\item El algoritmo se programo utilizando \citeasnoun{R} \dots

\end{itemize}

\section{Conversión a PDF (en Windows con WinEdt)}
Para conservar el formato de la página, la conversión de \emph{dvi}
a \emph{pdf} se debe hacer de la siguiente manera:
\begin{itemize}
\item Convertir \emph{dvi} $\rightsquigarrow$ \emph{ps}.
\item Abrir el \emph{ps} con el programa \emph{GSview}.
\item Convertirlo a \emph{pdf} con la opción \emph{pdfwrite} con
máxima resolución.
\end{itemize}

En los procesos de conversión puede suceder que algunas gráficas
tapen texto. Se debe observar el \emph{pdf} para verificar que eso
no ha sucedido. Si hay texto oculto se debe editar y dar el espacio
suficiente para que esto no ocurra.

\appendixsection{}\label{apenB}

In this appendix we derive an approximation of the variance of $\widehat{C}%
_{pmk}.$ Let $\{X_{t}\} $ be a stationary gaussian process. Writing the
estimator of $C_{pmk}$ as function of $\overline{X}$ and $S^{2},$
%
\begin{equation*}
\widehat{C}_{pmk}=\frac{a-\big|2\overline{X}-b\big|}{6\Big[S^{2}+\big(\overline{X}-T\big)^{2}\Big] ^{\frac{1}{2}}}=h\big(\overline{X},S^{2}\big)
\end{equation*}
%
we have,
%
\begin{equation*}
Var\big[\widehat{C}_{pmk}\big] \approx Var\big[S^{2}\big] \left\{\frac{\partial h( S^{2},\overline{x}) }{\partial S^{2}}\Big| _{\mu_{S^{2}},\mu _{\overline{X}}}\right\}^{2}
+ Var\big(\overline{X}\big) \left\{\frac{\partial h( S^{2},\overline{x}) }{\partial\overline{x}}\Big| _{\mu _{S^{2}},\mu _{\overline{x}}}\right\}^{2}
\end{equation*}
where
$\frac{\partial h(\overline{x},S^{2})}{\partial S^{2}}= \frac{a-| 2\overline{x}-b| }{6}\big[-\frac{1}{2}\big]\big[ S^{2}+(\overline{x}-T) ^{2}\big]^{-\frac{3}{2}}$
after some algebra, we obtain
%
\begin{multline*}
Var\big(\widehat{C_{pmk}}\big) \thickapprox C_{pk}^{2}\left[\frac{1}{f(n,p_{i})+\xi^{2}}\right] \times \\
\left\{\frac{F(n,p_{i})}{2(n-1)^{2}[f(n,p_{i})+\xi^{2}]^{2}}+\frac{g(n,p_{i})}{9n}\left[\frac{1}{C_{pk}}+\frac{6\xi}{2[f(n,p_{i})+\xi^{2}]}\right]^{2}\right\}
\end{multline*}
